# 🚖 Таксопарк "Ночной Амстердам"

Веб-приложение для управления автопарком с системой бронирования, админ-панелью и поддержкой загрузки фотографий автомобилей.

## 📋 О проекте

Полноценная система для таксопарка, которая позволяет клиентам просматривать доступные автомобили, бронировать их на нужное время, а администратору — управлять автопарком и обрабатывать заказы.

### 🔑 Основные возможности

#### Для клиентов:
- Просмотр всех доступных автомобилей с фотографиями
- Фильтрация по цене и количеству мест
- Бронирование автомобиля с указанием маршрута и времени
- Автоматический расчёт стоимости поездки
- Просмотр статуса бронирования

#### Для администратора:
- Защищённый вход в админ-панель (логин/пароль)
- Полное управление автопарком (добавление, редактирование, удаление машин)
- Загрузка фотографий для каждой машины
- Управление бронированиями (подтверждение, отмена, завершение)
- Статистика по бронированиям и автопарку
- Просмотр всех заказов с деталями клиентов

## 🛠️ Технологии

- **Backend:** Python 3.9+, Flask
- **База данных:** SQLite + SQLAlchemy ORM
- **Frontend:** HTML5, CSS3 (адаптивный дизайн)
- **Безопасность:** Хэширование паролей (SHA-256), сессии

## 🚀 Установка и запуск
### 1. Создание виртуального окружения
# Windows
python -m venv .venv
.venv\Scripts\activate

# macOS/Linux
python3 -m venv .venv
source .venv/bin/activate
### 2. Установка зависимостей
pip install flask flask-sqlalchemy
### 3. Запуск приложения
python main.py
### 4. Доступ к приложению
Сайт для клиентов: http://localhost:5000

Админ-панель: http://localhost:5000/admin/login

Учётные данные администратора по умолчанию:

Логин: admin

Пароль: admin123

⚠️ Важно: После первого входа смените пароль администратора!
